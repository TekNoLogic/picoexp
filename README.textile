picoEXP is a small "LibDataBroker":http://github.com/tekkub/libdatabroker-1-1/wikis based info addon.
It provides a simple EXP display, with a tooltip that shows detailed session experience statistics.

<em> picoEXP does not provide it's own display, you will need a LDB-based display addon. </em>
